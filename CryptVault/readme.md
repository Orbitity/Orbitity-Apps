# 🔐 CryptVault

CryptVault is a lightweight security utility designed to scan files and URLs for potential threats, suspicious patterns, and unsafe content indicators.

It provides a simple desktop interface and runs as a standalone Windows application powered by Electron.

---

## 🚀 Features

- 📁 File scanning for suspicious or unsafe content
- 🌐 URL scanning and validation checks
- ⚡ Fast local processing
- 🔐 Secure desktop environment
- 🖥️ Simple launcher-based UI
- 📦 Windows installer support (Inno Setup)

---

## 🧠 How It Works

CryptVault analyzes:
- File structure and content patterns
- Known unsafe signatures (rule-based detection)
- URL formatting and basic risk indicators
- Network safety checks (when enabled)

> Note: CryptVault is not a full antivirus replacement. It is a lightweight analysis and scanning tool.
